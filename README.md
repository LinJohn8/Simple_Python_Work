# 🐍 Simple_Python_Work - 我的Python小工具集合

> **一个收集了我各种Python小项目的宝库** 💎

欢迎来到我的Python小工具集合！这里存放了我日常开发中积累的各种实用Python脚本和工具，涵盖了自动化、数据处理、图像处理、Web开发等多个领域。

## 🎯 项目特色

- **简单实用** - 每个工具都力求简洁高效
- **即开即用** - 大部分脚本无需复杂配置
- **学习参考** - 适合Python初学者学习参考
- **持续更新** - 根据需求不断添加新功能

## 📁 项目结构说明

```
Simple_Python_Work/
├── 3D/                 # 3D图形与特效
├── Auto/               # 自动化工具

├── Draw/               # 绘图与图像处理
├── DynamicViewChart_csv/ # 数据可视化
├── GenerateQRcode/     # 二维码生成器
├── Local_AI_website/   # 本地AI工具
├── NovelDownload/      # 小说下载器
├── OtherFunction/      # 其他功能
├── PDFCreate/          # PDF文档生成
├── PngToIcon/          # 图标转换工具
├── QucikFunction/      # 快速操作工具
├── TTS/                # 语音处理
├── VideoAndAudio/      # 音视频处理
├── Web/                # Web工具
├── Widget/             # 界面工具
├── animation/          # 动画效果
├── simulation/         # 模拟功能
├── build/              # 构建输出
└── dist/               # 分发文件
```
